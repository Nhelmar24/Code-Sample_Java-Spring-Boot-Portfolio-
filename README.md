# Java Spring Boot Thymeleaf Portfolio

A simple, clean portfolio application built with **Java Spring Boot 3.x** and **Thymeleaf**. Demonstrates full-stack development skills with MVC architecture, responsive UI, and production-ready features.

## ✨ Features

- Dynamic profile pages with Thymeleaf templates and fragments
- Responsive design using Bootstrap
- Clean layered architecture (Controller → Service → Repository)
- Form handling and validation
- RESTful endpoints for portfolio data
- Unit tests with JUnit \& Mockito
- Docker containerization support


## 🛠 Tech Stack

```
Java 17+ | Spring Boot 3.x | Thymeleaf | Spring Data JPA
Maven | H2/MySQL | Bootstrap | Docker
```


## 🚀 Quick Start

```bash
# Clone & Run
git clone <your-repo>
cd spring-boot-portfolio
mvn clean install
mvn spring-boot:run

# Visit: http://localhost:8080
```


## 📁 Project Structure

```
src/main/
├── java/          # Controllers, Services, Entities
├── resources/
│   ├── templates/ # Thymeleaf HTML templates
│   └── static/    # CSS, JS, images
└── application.yml # Config
```
